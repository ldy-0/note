### JSON
轻量级数据交换格式。(全称JavaScript Object Notation) 
- 特点
1. 使用 JavaScript 语法来描述数据对象，但是它仍然独立于语言和平台。
2. 具有自我描述性，更易理解。
3. 体积小，易解析。
- 解析问题
1. 函数会被忽略。
2. 日期会被转为字符串。
3. 原型变为Object.prototype
### GeoJSON
用于编码各种地理数据结构的格式。
地理坐标系:WGS-84
